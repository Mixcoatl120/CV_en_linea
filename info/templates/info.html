Mas información del proyecto

Como mencione describiré las tecnologías para la creación de esta página sencilla. Empezare por el backend.El lenguaje que se encargara de todas las operaciones tanto de funciones y de alguno otro calculo para demostración es Python.

Este es un lenguaje de programación muy popular y versátil, se utiliza creación de programas de computadoras como la realización de aplicaciones web, automatizacion de tareas y analizis de datos .Incluso es el más usado en la industria para la cracion de IA's y Machine learning.

La versión de Python que esta ejecutando esta version web es Python 3.11.

1-Flask

El framework que se utiliza para que esta aplicación web pueda funcionar es el llamado FLASK.

¿Qué es Flaks?

Flask es un framework minimalista, el cual está basado en Python lo cual tiene como objetivo la creación de aplicaciones web bajo el patrón MVC(Modelo-Vista-Controlador). Como mencione este se basa en Python, pero también depende de un kit de herramientas WSGI de werkzeng, el motor de platillas jinja y el kit de CLIC

2-Bootstrap

Para los diseños estare usando bootstrap

Bootstrap es un conjunto de herramientas de código abierto para el diseño de aplicaciones web y sitios. Este framework multiplataforma contiene varias herramientas como botones, formularios, tipografías,etc.Esto facilitara a la hora de crear algun folmulario o una función para la página.
