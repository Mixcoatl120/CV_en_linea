¿Qué es JavaScript?

JavaScript es un leguaje de programación que se utiliza en el desarollo web ya que es fácil de hacer animacioes o acciones que le den vida a los sitios web y se han mas llamativos visualmente

para responder estas dudas haremos una pequeña explicación de lo que es este lenguaje cuales son la funciones de este encuanto a la programación web.

Introducción

JavaScript es un lenguaje de programación o de secuencias de comandos que te permite implementar funciones complejas en páginas web.

cada vez que una página web hace algo más que sentarse allí y mostrar información estática para que la veas, muestra oportunas actualizaciones de contenido, mapas interactivos, animación de Gráficos 2D/3D, desplazamiento de máquinas reproductoras de vídeo, etc.

Es la tercera capa del pastel de las tecnologías web estándar, dos de las cuales (HTML y CSS) hemos cubierto con mucho más detalle en otras partes del Área de aprendizaje.

Variables

Las variables son la forma de asignar los datos o valores, que usara el lenguaje para realizar operaciones algunos de estos son:

  • Numeros
  • String
  • Booleanos
Estructuras de control

Como sabemos los los lenguajes de programacion usan estructuras algunas de ellas son estructuras de ciclos:

  • forfor ([expresion-inicial]; [condicion]; [expresion-final]){sentencias}
  • While while (condicion){ sentencias}
  • Do-while do{ sentencia }while (condición);

Otras estructuras de control son las que tiene una funcion de seleccion estas son:

  • If If(condicion){ "instrucciones a seguir" } --------------------------
  • Swicht-Case switch (expresión) { case valor1: //Declaraciones ejecutadas cuando el resultado de expresión coincide con el valor1 [break;] case valor2: //Declaraciones ejecutadas cuando el resultado de expresión coincide con el valor2 [break;] ... case valorN: //Declaraciones ejecutadas cuando el resultado de expresión coincide con valorN [break;] default: //Declaraciones ejecutadas cuando ninguno de los valores coincide con el valor de la expresión [break;] }
